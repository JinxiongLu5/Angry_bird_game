Detailed Description

Class responsible for managing and displaying the game score, throwables left, enemies left, and the current level.

ScoreManager handles the game's scoring system, including tracking and updating the score and displaying game-related statistics such as the number of throwables left, enemies left, and the current level. It also manages the display of the game's result (win/lose).
